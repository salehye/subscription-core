# Subscription Core — التوثيق الكامل

[![PHP](https://img.shields.io/badge/PHP-^8.3-777BB4)](https://php.net)
[![Laravel](https://img.shields.io/badge/Laravel-^11.0|^12.0|^13.0-FF2D20)](https://laravel.com)

**حزمة Laravel قوية وقابلة للتوسّع لإدارة الاشتراكات** — تدير الخطط、الميزات、الاشتراكات、وتتبع الاستخدام مع دعم **تعدد المستأجرين (Multi-Tenancy)**. صمّمت لتكون مرنة、قابلة للتخصيص بالكامل、وجاهزة للتكامل مع أي نظام دفع.

---

## 🌟 لمحة عامة

هذه الحزمة تقدّم نظامًا متكاملًا لإدارة الاشتراكات في تطبيقات Laravel. صُمّمت **بدون بوابات دفع** (No payment gateway) مما يتيح لك ربطها مع أي نظام دفع تريده (Stripe、PayPal、Moyasar、Tabby... إلخ).

### الميزات الرئيسية

| الميزة                    | الوصف                                                                     |
| ------------------------- | ------------------------------------------------------------------------- |
| ✅ **إدارة الخطط**        | إنشاء خطط بدورة فاتورة شهرية、سنوية、أو مدى الحياة                        |
| ✅ **الميزات (Features)** | 3 أنواع: Toggle (تشغيل/إيقاف)、Consumable (قابلة للاستهلاك)、Limit (حدود) |
| ✅ **الاشتراكات الكاملة** | اشتراك、إلغاء、تجديد、تبديل خطة、إيقاف مؤقت、استئناف                      |
| ✅ **الإضافات (Add-ons)** | إضافة خدمات إضافية للاشتراك الأساسي                                       |
| ✅ **تتبع الاستخدام**     | تتبع استهلاك الميزات مع تحديد تلقائي للحدود                               |
| ✅ **دمج الميزات**        | جمع قيم الميزات من الاشتراك الأساسي + الإضافات تلقائيًا                   |
| ✅ **تعدد المستأجرين**    | نطاق لكل مستأجر (SaaS)                                                    |
| ✅ **نظام الأحداث**       | أحداث غنية لكل دورة حياة الاشتراك                                         |
| ✅ **أوامر Artisan**      | لإنهاء الاشتراكات、إعادة تعيين الاستخدام、توليد الفواتير الدورية          |
| ✅ **ميدلوير**            | حماية المسارات بناءً على الميزات                                          |
| ✅ **Polymorphic**        | تعمل مع أي موديل (User、Team、Organization...)                            |
| ✅ **قابلية التوسع**      | استبدال أي مكون عبر العقود (Contracts)                                    |
| ✅ **بدون فواتير**        | ترك المجال لنظام الدفع الخارجي                                            |
